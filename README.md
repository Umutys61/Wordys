# 🧩 Wordys

A Wordle-style word puzzle game developed using **Unity** and **C#**.  
The goal of the game is to guess the hidden word within a limited number of attempts using a grid-based system.

This project was developed as part of my game development learning journey.

---

## 🎮 Gameplay

The player tries to guess the correct word by entering letters into a grid.  
After each guess, the game provides feedback about which letters are correct.

### Features

- Grid-based word guessing system
- Keyboard input system
- Hint system using in-game coins
- Classic game mode
- Timed game mode
- UI feedback for correct / incorrect letters

## 🚀 How to Run the Project

1. Clone the repository

2. Open **Unity Hub**

3. Click **Add Project**

4. Select the project folder

5. Open it with a compatible Unity version
